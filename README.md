# ⚡ Bolt.newer - Sandbox AI Coding Environment

**Bolt.newer** is a full-stack developer workspace application that mimics sandbox IDEs. It allows developers to prompt an LLM, dynamically generate file edits, and render/execute web applications in real-time inside the browser container.

---

## 📦 System Architecture

### 1. Frontend (`/frontend`)
* Built with **Vite, React, and TypeScript**.
* Styled using TailwindCSS and styled layouts featuring Lucide React Icons.
* Integrated with **Monaco Editor** for fully functional, syntax-highlighted in-browser file editing.
* Uses **@webcontainer/api** to provision and execute local development servers in the browser sandbox.

### 2. Backend (`/be`)
* Lightweight **Express API** written in TypeScript.
* Integrates the **@anthropic-ai/sdk** to stream and structure code modifications generated by Claude models.

---

## 🛠️ Technology Stack

| Layer | Tools |
| :--- | :--- |
| **Frontend** | Vite, React 18, TypeScript, Monaco Editor, Lucide React |
| **Web Container** | `@webcontainer/api`, Axios, XML2JS |
| **Backend Server** | Node.js, Express, TypeScript |
| **LLM Provider** | Anthropic AI API (Claude SDK) |

## 🚀 Getting Started

### Prerequisites
Ensure Node.js is installed.

### Setup Instructions
1. Navigate to the backend directory and set up:
   ```bash
   cd be
   npm install
   ```
   Configure the Anthropic API key inside a local `.env` file:
   ```env
   ANTHROPIC_API_KEY=your_key_here
   PORT=5000
   ```
   Run backend server:
   ```bash
   npm run dev
   ```

2. In another terminal, navigate to the frontend directory:
   ```bash
   cd ../frontend
   npm install
   npm run dev
   ```
3. Open your browser to the URL printed by Vite to access the sandbox coding interface!
